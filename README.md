## Опис проєкту

**Вебсайт "Ботанічний довідник"** — 3 взаємопов'язані сторінки з семантичною розміткою HTML5, єдиним CSS-файлом стилів, формами з валідацією, медіаелементами (аудіо, відео), iframe та інтерактивною навігацією.

Виконано всі 3 блоки завдань (оцінка 5):

| Блок | Зміст | Статус |
|------|-------|--------|
| 1 (базовий) | Семантична розмітка, мета-теги, CSS-стилі, карта сайту | Виконано |
| 2 (розширений) | Випадаюче меню, iframe, аудіо/відео, форми з валідацією | Виконано |
| 3 (просунутий) | Градієнти, тіні, прозорість, псевдокласи, дочірні селектори | Виконано |

## Запуск

```bash
git clone https://github.com/oleksandrkonovalov1/hypertext-lab1.git
cd hypertext-lab1

# Відкрити в браузері
open src/index.html

# Або запустити локальний сервер
npm run serve
```
